** Usage

#+BEGIN_SRC sh
  # cd your-blog
  git clone https://github.com/loveminimal/hexo-theme-zero.git ./themes/zero
#+END_SRC

Note that: you must disable the =highlight= of site =_config.yml=, and update =theme= fragment as =zero=.

#+BEGIN_EXAMPLE
  # highlight settings
  highlight:
    enable: false
    line_number: false
    auto_detect: false
    tab_replace:

  # theme settings
  theme: zero
#+END_EXAMPLE
